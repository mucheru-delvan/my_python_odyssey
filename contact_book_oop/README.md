

#  Python Contact Book 

A simple and interactive **Python Contact Book** that helps you save, search, update, and manage your personal contacts!
All your data is automatically stored in a JSON file so your contacts are always safe.

---

## Description

This program allows you to:

* Add new contacts with a **name** and **phone number**
* Automatically **update** a number if the contact already exists
* Search for any saved contact
* Remove contacts by name (case-insensitive)
* View all stored contacts in a clean, readable list
* Automatically save and load data using a **JSON file**
