
# Even & Odd Number Counter

A simple and fun **Python command-line program** that counts how many even and odd numbers exist in a range you choose.
A great beginner project to understand loops, conditionals, and user input in Python.

---

## Features

* Enter your own **start** and **end** numbers.
* Automatically counts how many numbers in the range are **even** and how many are **odd**.
* Clean and easy-to-read output.
* Quick to run â€” works in any terminal with Python installed.

## Example

```text
Even & Odd Counter

Enter start number: 1
Enter end number: 10

Even numbers from 1 to 10: 5
Odd numbers from 1 to 10: 5
```
