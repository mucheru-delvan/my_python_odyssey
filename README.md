## 🚀 Getting Started

Clone this repository:
```bash
git clone https://github.com/mucheru-delvan/my_python_odyssey.git
cd my_python_odyssey
```

Navigate to a project folder and run:
```bash
python3 main.py
```

Each project includes its own README with:
